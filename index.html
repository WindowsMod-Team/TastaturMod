Tastatursteuerung Komplett ohne Tastatur-Hardware.

Das TastaturMod Tool wurde speziell für den Office-Einsatz und leicht-Gaming entwickelt. Sie ermöglicht das ununterbrochene, hochfrequente Senden von Tastaturbefehlen über eine minimalistische, rahmenlose Oberfläche.

✓ Hochfrequentes Signal-Intervall: Die Anwendung arbeitet ununterbrochen mit einer präzisen Taktung von exakt 15 Millisekunden im Hintergrund.
✓ Unsichtbare Gestensteuerung: Keine störenden Icons in der Taskleiste – das Interface wird komplett flexibel über die Maus gesteuert.
✓ Nahtlose Windows-Integration: Ein virtuelles Overlay garantiert die dauerhafte Funktion im Hintergrund. Parallele Richtungseingaben sind jederzeit möglich.

Dokumentation der Steuerung

Funktion Tastaturbelegung (Maus)
Bildschirmtastatur EIN / AUS Mausrad-Doppelklick
Steuerungs-Box EIN / AUS Rechts-Doppelklick
Dauerklick aktivieren Eingabe im Textfeld + Klick auf OK
Dauerklick deaktivieren Klick auf Stopp
Software-Paket herunterladen (.ZIP)
Geprüfte Signatur. Kompatibel mit Windows 10 und Windows 11 (64-Bit).
